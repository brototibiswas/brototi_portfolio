Capstone: Fieldwork Application Website

I participated in this Capstone project as a Fullstack Web Developer. I collaborated in a team of 2 with our faculty supervisor/professor to work on the Fieldwork Application website for the SUNY New Paltz Career and Resource Center. This was a complex, sophisticated system built by previous Capstone participants and it was in production for a few semesters. In this semester, me and my teammate worked on debugging the entire system so that the application process flows smoothly and accurately. We also changed to flow of the application so it was more user-frieldly. Technologies used: PHP, HTML, Bootstrap, Javascript, and MySQL for database.

Our key achievement was implementing the Admin panel with features such as - Drag-and-drop Application Sequence Editor, viewing all the application on system, creating new users and departments.

The drag-and-drop feature was implemented using vanilla Javascript. The color theme closely followed the SUNY New Paltz Brand Identity Guideline.

Initially, the secretary would create the new application by inputting only the student ID and the student would complete the entire application, including student and employer information, fieldwork learning objective, fieldwork descriptions, and course information.

We edited the application flow so that the secretary would fill the course information and the learning objectives are added by the instructor and the employer. This makes the student application more straight forward and easy to complete.

I implemented feature to show all the applications in the database. This was done by PHP in the backend to query the data. The frontend is styled using Bootstrap.
